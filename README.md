# 5G MIMO-OFDM System Simulation in MATLAB

This repositoty contains a MATLAB-Based simulation of a 5G-inspired MIMO-OFDM Wireless communication system.

## System Overview
- SISO OFDM system
- Alamouti Space-Time Block Coding (2×1 and 2×2)
- Rayleigh fading and AWGN channel models
- Pilot-based channel estimation using Least Squares (LS)
- Zero Forcing (ZF) and MMSE equalization
- BER vs SNR performance analysis

## Tools Used
- MATLAB

- ## Motivation
- This project was developed to strengthen understanding of wireless physical layer concepts and MIMO diversity techniques.
